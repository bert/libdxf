Introduction.

The group codes described in this chapter are found only in DXF files.
The CLASSES section holds the information for application-defined classes whose instances appear in the BLOCKS, ENTITIES, and OBJECTS sections of the database.
It is assumed that a class definition is permanently fixed in the class hierarchy.
All fields are required.

AutoCAD registers the classes listed in the following table.

Note: This may not be a complete list of the classes found in a DXF file.
It depends on the applications currently in use by AutoCAD.
